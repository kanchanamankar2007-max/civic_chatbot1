# AI-Powered Multilingual Civic Services Copilot

## 📌 Project Description
This is an AI chatbot that helps users understand government schemes easily.

## 🚀 Features
- Multilingual support
- AI-generated responses
- Simple and user-friendly interface

## 🧠 How it works
User → Translate → Search data → AI → Answer

## ▶️ How to run
pip install streamlit pandas openai googletrans  
streamlit run app.py
